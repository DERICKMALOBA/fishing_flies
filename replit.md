# Premium Fishing Flies E-commerce Website

## Overview
A fully responsive, SEO-optimized e-commerce website for selling fishing flies, built using HTML5, Bootstrap 5, CSS3, and vanilla JavaScript. The website features an aquatic/fly-fishing aesthetic with blue, teal, aqua, and white tones.

## Project Structure
```
/
├── index.html          # Home page
├── contact.html        # Contact page
├── css/
│   └── styles.css      # Main stylesheet with aquatic theme
├── js/
│   └── main.js         # JavaScript for search, WhatsApp integration, lazy loading
├── dry-flies.html      # Dry Flies category
├── parachutes.html     # Parachute Flies category
├── nymphs.html         # Nymphs category
├── emergers.html       # Emergers category
├── midges.html         # Midges category
```

## Features
- 21 pages total (home, contact, 19 category pages)
- Responsive mobile-first design
- SEO-optimized with proper meta tags and heading hierarchy
- Client-side search functionality
- WhatsApp ordering integration (floating button on all pages)
- Clean URLs for SEO
- Bootstrap 5 via CDN for fast loading
- Font Awesome icons via CDN
- No backend required - static site

## Technical Details
- **Frontend**: HTML5, Bootstrap 5.3.2, CSS3, Vanilla JavaScript (ES6+)
- **Icons**: Font Awesome 6.4.2
- **Styling**: Custom CSS with CSS variables for theming
- **Server**: Python HTTP server for development (port 5000)

## Key Design Elements
- Aquatic color palette (blues, teals, aquas)
- Card-based product layouts
- Smooth hover effects
- Floating WhatsApp button with pulse animation
- Gradient headers and CTAs
- Mobile-responsive navigation

## Running the Project
The site is served using Python's built-in HTTP server on port 5000.
